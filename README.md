Basic implementation of Word2Vec in Python.

This project contains 3 files:
1. model.py: The core word2vec program which assigns vectors to words
2. utils.py: Preparing the data to be analyzed by the model
3. train.py: Script to train the model and a short test with sample words

The dataset used to train the model is given in text8.
Similarity calculation technique: Skip-gram with negative sampling.
